# 🧒 儿童学习小达人

一款面向小学1-6年级学生的互动学习App，涵盖**数学**、**语文**、**英语**三大学科，内置宠物养成、金币奖励、闯关挑战等趣味系统，让孩子在快乐中学习成长！

## ✨ 功能特色

### 📚 三大学科
- **数学** — 加减乘除、混合运算、应用题，支持自由练习和闯关模式
- **语文** — 拼音、汉字、词语、成语、古诗、阅读理解，按年级分层
- **英语** — 单词、短语、句型、语法、阅读，匹配小学各年级水平

### 🎮 多种练习模式
- **闯关模式** — 从数字启蒙到复杂运算，层层递进解锁
- **自由练习** — 自选年级和题型，灵活刷题
- **速度挑战** — 限时答题，锻炼反应速度

### 🐾 宠物养成系统
- 选择并喂养你的专属学习伙伴
- 宠物随学习进度一起成长
- 解锁更高等级的挑战内容

### 💰 金币奖励体系
- 做题获得金币奖励
- 连续登录领取奖励
- 暴击机制获取额外金币

### 📊 学习统计
- 记录每日学习数据
- 正确率、答题数量等多维度统计
- 可视化学习进度

## 🛠 技术栈

| 技术 | 说明 |
|------|------|
| **Next.js 16** | React 框架 (App Router) |
| **TypeScript** | 类型安全 |
| **Tailwind CSS 4** | 样式框架 |
| **shadcn/ui** | UI 组件库 |
| **Zustand** | 状态管理 |
| **Prisma** | 数据库 ORM (SQLite) |
| **Framer Motion** | 动画效果 |
| **Lucide Icons** | 图标库 |

## 🚀 快速开始

```bash
# 安装依赖
bun install

# 初始化数据库
bun run db:push

# 启动开发服务器
bun run dev
```

打开浏览器访问 `http://localhost:3000` 即可使用。

## 📁 项目结构

```
src/
├── app/                    # Next.js App Router 页面
│   ├── page.tsx           # 主入口
│   ├── layout.tsx         # 全局布局
│   └── api/               # API 路由
├── components/
│   ├── math/              # 数学模块（主页、答题、统计等）
│   ├── chinese/           # 语文模块
│   ├── english/           # 英语模块
│   ├── pet/               # 宠物系统
│   ├── shared/            # 共享组件
│   └── ui/                # shadcn/ui 组件
├── lib/
│   ├── game-store.ts      # 全局状态 (Zustand)
│   ├── math-utils.ts      # 数学题目生成
│   ├── chinese-utils.ts   # 语文题库
│   ├── english-utils.ts   # 英语题库
│   ├── achievements.ts    # 成就系统
│   └── db.ts              # 数据库客户端
├── hooks/                 # 自定义 Hooks
└── app/
    └── globals.css        # 全局样式
```
